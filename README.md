# 筋トレ記録アプリ

トレーニングセッションを記録・管理するAndroidアプリです。

## 機能

- セッション記録（種目・セット・重量・REP・RIR・没頭度・メモ）
- セッション一覧（月・種目フィルター）
- セッション詳細・編集・削除
- 種目マスター管理

## 技術スタック

| 役割             | 技術            |
| ---------------- | --------------- |
| UIフレームワーク | Flutter         |
| 状態管理         | Riverpod        |
| DB               | Drift（SQLite） |
| ナビゲーション   | go_router       |

## 開発環境

- Flutter 3.x
- Dart 3.x
- Android Studio / VS Code
